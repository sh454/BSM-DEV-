# BSM-DEV-
Bu proje, belirtilen bir dizindeki değişiklikleri (dosya oluşturma, silme, değiştirme) izleyen bir Python servisini içerir. watchdog kütüphanesi kullanılarak tespit edilen değişiklikler JSON formatında bir log dosyasına kaydedilir. Servis, Linux ortamında systemd ile otomatik olarak başlatılacak şekilde yapılandırılmıştır.SHAHRYAR HAYATI
